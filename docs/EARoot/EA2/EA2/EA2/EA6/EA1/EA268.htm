Registreren applicatie [KT.UC.102]
Als IT-deelnemer wil ik dat de systeembeheerder van de Koppeltaal voorziening mijn applicatie registreert in het stelselregister. Deze registratie heb ik nodig om via mijn applicatie Koppeltaal functionaliteit te kunnen ontsluiten.<br/>
